TTS Reader
- Text to Speech Reader App

Results

By using TTS Reader, users free their hands and eyes, because the app primarily focused on listening to documents, with reading as a supplement method, enabling users to engage with written content through auditory means.

TTS Reader supports crucial functions such as creating new TXT files, importing documents, scanning images, and downloading .txt files via browsing, effectively meeting the users' requirements and preferences.

The interface is designed to be concise and intuitive, allowing individuals of all ages, including children and elderly, to swiftly access essential features.

Overview

TTS Reader is an app specifically designed for listening to .txt files. It offers 4 text import options and supports multiple languages for listening to .txt files. Moreover, it provides a user-friendly text reading interface, allowing users to easily follow along with the text.

With its straightforward functionality, TTS Reader is an accessible and user-friendly app for listening to txt files.

Design

Home Page

Basic Features
The basic features include document search, adding new files, layout options, and view modes.

The default layout option is Sort by Date and the default view mode is Column View.

The reason I chose not to implement Search and Add New File as pop-up windows is to ensure a seamless and fluid user experience, maintaining the overall visual coherence of the page.

Supported File Formats
Supported file formats include .txt, .zip, and general files.

Single Selection
When a file is selected, the following options will be available under the file: Select All, Move, Rename, Edit, Pin, Share, Compress, and Delete.

Multiple Selection
When selecting multiple files, the options available are: Select All, Move, Compress, and Delete.

If undoing the selection, the options will appear under the previously selected file.

4 Text Import Options

Create New Text File
Users can create a new .txt file and start typing by simply tapping on the first row in the column view mode or the first box in the gallery view.

Open File
Users can import their own files by tapping the plus button and selecting "Open File".

Scan
Users can choose to scan images to extract text and create their .txt files. They have the option to take single shots or multiple shots when capturing images. Of course, users can import photos from album as well.

Browse
Users can download .txt files through "Browse" option, and the default search engine for browsing is set to Google Search.

Audio Page

Basic Features
There are two available display modes: "Light Mode" and "Dark Mode".
The "Search" feature allows users to search for content within the entire file.

Play
When entering a .txt file, the text will automatically start playing from the beginning of the paragraph where the user last left off. Alternatively, users can tap on any paragraph within the text interface to start playing from the beginning of that specific paragraph.

Users have the flexibility to customize the number of paragraphs displayed per page based on their preferences. When tapping the "Play Next" button, it will play the content of the next page directly.


Page
Users can navigate to any page within the text interface, and it will start playing from the beginning of the selected page.

Chapter
Additionally, users have the option to choose and play a specific chapter.

Voice
Users can choose their preferred voice from multiple options available for each language. They can also adjust the rate and pitch of the voice. Additionally, special characters inputted will be skipped while playing.

Timer
Users can set a timer, and the play will stop when the timer runs out.

Conclusion

I'm interested in designing a TTS Reader for two reasons. Firstly, AI voice technology has advanced, making AI-generated speech more natural and acceptable. Secondly, using this app allows individuals to free up their hands and engage in other tasks. Whether it is listening to books, manuscripts, news, or educational content, all of these activities involve input and can deepen a person's impression.

Therefore, I aim to create a user-friendly interface that is simple and practical, enabling people of all age groups to quickly and easily utilize the app.

Gains

During my work on the TTS Reader, I have learned the importance of receiving user feedback. It makes me to gain a deeper understanding of users' precise needs and experiences while using an app, which in turn helps me refine and improve my design even further.
